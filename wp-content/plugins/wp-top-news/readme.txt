This plugin works in two ways - internal and external news sources.

You can simply add your news in your WordPress admin panel and display them with a shortcode [wtn_news layout='list/grid'].

Another way is integrate this plugin with an API collected from newsapi.org and display news from various news sources and countries.

TopNewsWp is a simple, effective and powerful plugin to display your desired news.

== Installation ==

Install Worldwide Breaking News Headlines as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search WP Top News
3. Click to install
4. Activate the plugin
5. Setup the proper settings in WP Top News menu

**Install by upload plugin:**

1. Download the latest version of WP Top News (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded wp-top-news.zip and click on "Install Now" button
6. Activate the plugin

== Frequently Asked Questions ==

= How can I Display the News? =

Internal News:

After activating the plugin, you will see "News‚Äù in the Admin Dashboard Menu.

Go to ‚ÄúAdd New Item‚Äù and add your News as many as you want.

Now you need to insert the shortcode [wtn_news layout='grid/list'] at any page through TinyMCE editor. 

External News:

After activating the plugin, you will see ‚ÄúTop News‚Äù menu in WordPress Admin Dashboard.

Firstly, Go to ‚ÄúAPI Settings‚Äù and add your API which you collected from newsapi.org

Secondly, Go to "General Settings" and select your news source.

Finally, Put the shortcode [wp_top_news] in a page editor and Update it.

You will see your news section on that page.

Note: We can not guarranty the API working or not because we have no relation with them. So buy the API at your own risk!

= I have changed News source, but news feed not uppdating =
As there is a cache active for 15 minute, so you have to wait for 15 minute to see the updated news.

= How can I reduce the Cached time? =
The API we are using here has a limitation for 15 minute delay news update. So there is nothing we can do now.

= How to increase requests per day for a specific API Key? =
You can register to NewsAPI.org and get your personal API key and use it in API Key field. This will increase your requests per day limit.

Note: We can not guarranty the API working or not because we have no relation with them. So buy the API at your own risk!

= I have changed News source but there is no data! =
Sometimes due to main API server loading issue you may see no data. Keep refreshing until your data is properly cached.

= Any planning to add more options/newspaper? =
Yes, we are working on it. Please stay with us.
